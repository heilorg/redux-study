# redux-study

### velopert 블로그에서 배운 리액트 리덕스

출처 : https://www.youtube.com/watch?v=xX0eQLQ3IJw&list=PL9FpF_z-xR_GMujql3S_XGV2SpdfDBkeC&index=24

> 주요기능

* 리액트
* 리액트 리덕스

> 사용법

의존성 설치

```bash
npm install
```

리액트 프로젝트 실행

```bash
npm start
```

1. (+) 버튼을 누르면 숫자가 1 증가한다
2. (-) 버튼을 누르면 숫자가 1 감소한다
3. Random Color 버튼을 누르면 배경색이 랜덤한 색으로 지정된다.
